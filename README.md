fancyogame
==========

More interesting than ogame


Design
===============

特性
1. 更为丰富的科技、建筑和舰船
1.1 登陆舰队，允许战斗中俘获地方舰船
2. 科技的影响范围大大加强（战斗策略，能源利用效率）
3. 不同类型的武器有各自的效果
4. 星球能量系统的改进（能量槽，更好的负载管理与收益）
5. 舰队编队
6. 更好的航道管理，区分攻击和其他任务
7. 基本资源可瞬间远距离传递（科技支持）
8. 去除星球方圆概念，引入资源总储量概念
9. 可投放采矿模组到无人星球上。采矿模组自带供能和资源传输系统，能将所采资源传输到预设星球上
10. 采矿模组可被攻击舰队摧毁，且无法重建

10. 统一货币为银河币（GC），并开放交易市场
11. 目前允许的交易对象为资源，舰船，模组
12. 每个银河系有若干个中继器，中继器之间可以进行瞬间传送
13. 使用中继器需要购买许可证，并需按舰队规模缴纳银河币
14. 舰队经过中继器会被记录，可在黑市购买舰队通过中继器的记录
15. 每个星球可建立一个贸易港口，根据星球的资源产量和使用量，每天会产生一定的银河币收入

16. 战争学院，提供对战争的高级辅助，游戏后期的核心
16.1 部分战舰有能力俘获地方飞船
16.2 大中型战舰能搭载多种武器，并依战场形势灵活使用
16.3 多样化的战斗策略，以少胜多成为可能
